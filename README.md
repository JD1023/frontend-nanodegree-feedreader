## About

* This project is part of the Udacity Front-End Web Development Nanodegree.
* The Goal of this project is to run a number of test on a pre-existing RSS Feed Reader application using Jasmine.

## Instructions

* Download or clone the project files from this repository. Run the index.html file in your browser.
* Scroll down to the bottom of the page to see the Jasmine test results.

## Dependencies

* [jasmine-2.1.2](https://github.com/jasmine/jasmine)
* [handlebars js](http://cdn.jsdelivr.net/handlebarsjs/2.0.0/handlebars.min.js)
* [jquery](http://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js)
* [jsapi](http://google.com/jsapi)
